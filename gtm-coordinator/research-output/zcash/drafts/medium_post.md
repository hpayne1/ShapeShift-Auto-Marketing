### What Is Zcash?

Zcash launched in 2016 with a UTXO-based architecture similar to Bitcoin, adding zk-SNARKs â€” zero-knowledge proofs that allow transaction details to be verified without revealing them. This made Zcash the first production blockchain to implement zero-knowledge cryptography, giving users the choice between transparent and shielded transactions.

Zcash supports two address types:

- **Transparent addresses (t-addresses):** Similar to Bitcoin â€” transactions are visible on-chain.
- **Shielded addresses (z-addresses):** Encrypted using zk-SNARKs â€” amounts, senders, and receivers stay private.

ShapeShift currently supports transparent (t-address) transactions.
