# Release Notes — Zcash (ZEC) Support

## What's New

**Zcash (ZEC) support — send, receive, swap**

ShapeShift now supports Zcash (ZEC), the original privacy coin built with zk-SNARKs. Users can send, receive, and swap ZEC using transparent t-addresses, all from a self-custodial wallet.

### Details

- **Asset:** ZEC (Zcash)
- **Chain:** Zcash (UTXO-based)
- **Capabilities:** Send, Receive, Swap
- **Address type:** Transparent (t-addresses) only
- **Shielded (z-addresses):** Not supported at this time
- **KYC:** None
- **Custody:** Self-custodial — users hold their own keys

### Get started

Visit [app.shapeshift.com](https://app.shapeshift.com) to send, receive, or swap ZEC today.
