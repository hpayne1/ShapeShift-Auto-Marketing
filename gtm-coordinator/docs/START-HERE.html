ShapeShift Auto-Marketing — How It Works

One-pager: workflow, skills, and links

Workflow Tree

Phase 1: PLANNING gtm-workshop → PR link, protocol URLs (website, Twitter, Farcaster), transcript Phase 2: GENERATE gtm full-packet --protocol <Name> --url <URL> [--pr <PR>] [--protocol-twitter <X>] [--protocol-farcaster <FC>] Phase 3: GAP RESOLUTION gtm-gap-auditor → gtm-questionnaire-generator → product fills form → gtm-questionnaire-merger Phase 4: FINAL CHECK & LAUNCH gtm-final-check → Execute (checklist)
