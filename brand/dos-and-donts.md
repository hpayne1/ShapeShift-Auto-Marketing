- Don't skip deep links when they're available

### Frequency Rules
- **Memes:** Max 1x/week from main (comments section is fair game)
- Don't spam engagement posts
- Don't over-post without substance

### Topics to Avoid
- Price predictions or speculation
- Financial advice of any kind
- Competitor bashing
- Political/controversial topics
- Unverified claims
- KYC promotion (we're no-KYC)

## Quick Checklist Before Posting

### From Main Account:
- [ ] 3rd person voice (not casual "we")
- [ ] No faces in the post
- [ ] Clean, value-focused graphic
- [ ] Deep link included if applicable
- [ ] Contributor QT lined up for major launches
- [ ] Not a meme (unless it's the 1x/week allowance)

### For Contributor QT:
- [ ] Adds genuine context or walkthrough
- [ ] Personal perspective is welcome
- [ ] Face/Loom ok
- [ ] Ready to post when main drops

### For Data Posts:
- [ ] Real metrics (not vanity numbers)
- [ ] Simple visualization
- [ ] Not overdesigned
- [ ] Insight or context included

### For Engagement:
- [ ] Genuine question
- [ ] Easy to respond to
- [ ] Drives participation
- [ ] Not empty "GM" vibes
